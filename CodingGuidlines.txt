Guidlines HCI:

-Alles Englisch (GUI + Code)

-Member (globale) Variablen GROSS anfangen: z.B. private int MyFuckingId

-final / static / const Variablen UPPER CASE: z.B. private final int APIKEY

-Aussagekräftige Bezeichnung der Variablen und Methoden!!

-Einteilungen in Packages

Ganz wichtig:
Keine Commits (Push), mit Compiler Errors
Bei großeren Änderungen kurze Nachricht, was geändert wurde.
